# CRUD_JONS_ELETRICA

Sistema de gerenciamento de serviços elétricos para empre Jons Elétrica com funcionalidades CRUD (Create, Read, Update, Delete) e geração de orçamentos.

## Funcionalidades

- **Listar Serviços**: Exibe todos os serviços cadastrados com seus respectivos IDs e valores.
- **Criar Serviço**: Adiciona novos serviços ao sistema.
- **Orçamento**: Seleciona serviços para gerar um orçamento final com o valor total.
- **Atualizar Serviço**: Modifica o nome ou valor de um serviço existente.
- **Deletar Serviço**: Remove serviços do sistema.

## Como usar

1. Execute o arquivo `crud_jonseletrica.py`.
2. Navegue pelo menu utilizando os números.
3. Siga as instruções na tela para cada operação.

## Requisitos

- Python 3.x instalado

## Estrutura do Projeto

- `crud_jonseletrica.py`: Código principal do sistema.
- `servicos.json`: Arquivo JSON onde os dados são armazenados.
